# tuff

A Roblox exploit & Luau learning hub. Static single-page site covering:

- **Learn Luau** — 14 lessons from "hello, world" through metatables, coroutines, and the Roblox API.
- **Exploit Basics** — what an executor is, how injection works, the global vocabulary (`getgenv`, `hookfunction`, `__namecall`, etc.).
- **Executor Status** — tracked board of which executors are UP, DOWN, or DMCA'd, filterable by platform.
- **Scripts** — annotated working examples (Infinite Jump, ESP, Anti-AFK, remote spy hook).
- **FAQ** — bans, executors vs. scripts, DMCA, paid vs. free.

Educational. Modifying the Roblox client breaks ToS — read the warning on the page.
